# Guia do administrador — Fermo Studio
Acesse `/app/fermo-studio` (perfil admin).
- **Modelos**: catálogo, preço-base, prazo, mínimo, malhas editáveis, 3D, destaque.
- **Opções**: materiais, solados, forros, palmilhas, cadarços, embalagem, personalizações
  (preço por par/percentual/desenvolvimento, prazo adicional, mínimo).
- **Precificação**: perfil + versão (margem, custos de operação, perdas, impostos, sinal%,
  validade) e faixas de quantidade. Versões antigas preservam orçamentos já emitidos.
- **Projetos/Orçamentos**: acompanhamento do funil e conversão em pedido.
